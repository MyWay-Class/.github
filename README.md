# MyWayClass Organization Hub

이 레포는 조직 프로필과 협업 규칙을 함께 보여주는 진입점입니다.

- 조직 프로필에서 보이는 소개는 [`profile/README.md`](./profile/README.md) 를 사용합니다.
- 레포 페이지에서 바로 보이는 소개는 이 파일을 사용합니다.

## 한 줄 소개

강의를 “소비”가 아니라 “재구성”하는 AI 교육 플랫폼.

## 핵심 내용

- 강의를 STT로 텍스트화
- 의미 단위로 청킹
- RAG로 근거 구간 검색
- AI로 요약, 퀴즈, 질문 응답, 숏폼 생성
- 원본 강의와 연결된 학습 흐름 유지

## 협업 원칙

- 문서는 먼저 읽고, 코드는 그 다음에 만집니다.
- 지시는 짧게 쓰되, 범위와 완료 기준은 분명히 씁니다.
- 변경 후에는 `docs/dev-logs/`에 판단 근거를 남깁니다.

자세한 소개는 [`profile/README.md`](./profile/README.md) 를 확인하세요.
